# Reformer

PyTorch original implementation of **Neural Machine Translation with Joint Representation**. It is modified from [fairseq-0.6.0](https://github.com/pytorch/fairseq).

## Requirements

- PyTorch version == 1.1.0
- Python version == 3.6.7
